#sends_invoice_email(invoice, emails = nil) ⇒ void

This method returns an undefined value.

Sends an invoice to the emails provided using InvoiceMailer.

This method triggers the sending of an email with an invoice attached using the InvoiceMailer. It queues the email to be sent asynchronously for later delivery.

Parameters:

  • invoice (Invoice) —

    The invoice object to be included in the email

  • emails (Array<String>, nil) (defaults to: nil) —

    (Optional) An array of email addresses to send the invoice email to
